# IPO-Data-Analysis
This project analyzes IPO datasets to understand company listing performance and market trends.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Project Overview
The project performs exploratory data analysis (EDA) on IPO datasets to identify patterns in IPO performance and financial metrics.

## Key Features
- Data cleaning and preprocessing
- Exploratory data analysis
- Data visualization of IPO trends
